# Project Overview
This project implements Class-D audio amplifier
I here used IC555, op-amps, MOSFET, LC LPF, speaker

# Audio Input by Op-Amp-1
It amplifies low-level audio and gives proper voltage criteria for comparator

# PWM generation using IC555

# Comparator using Op-Amp-2
Compares audio vs carrier and produces PWM signal

# MOSFET Stage
Converts PWM to high-power switching and drives LC LPF

# LC LPF output
Removes HF PWM and passes only audio frequencies

# Power Supply
NE555:5V
Op-Amps:12V
MOSFET:12 V
